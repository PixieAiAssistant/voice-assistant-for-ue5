FAQ

Does Pixie work without Unreal Engine?

Yes. The Free version works as a full Windows voice assistant on its own — no UE5 project required.

Do I need to create an account?

No registration required. After payment, your license key is emailed to you automatically.

Is there a free trial for Pro?

There is no trial period. The Free tier itself is fully functional for Windows control — Pro is a paid upgrade for UE5 features.

Can I use my license on multiple machines?

Each license is bound to one machine (via a hardware ID). If you switch PCs, contact support via email to reset your device.

Which Unreal Engine version is supported?

Pixie Pro is built and tested for UE 5.8. Other 5.x versions may partially work, but are not officially supported.
